本组件是一个基于 React 的确认框组件。

## 使用

### 显示 Modal

```
Modal.show({
  header: '', // 黑字提示
  desc: '', // 灰色小字补充
  ok: '', // 确认按钮文字，默认为“确定”
  onOk: () => {}, // 确认按钮回调。无则不显示确认按钮
  cancel: '', // 取消按钮文字，默认为“取消”
  onCancel: () => {}, // 取消按钮回调。无则不显示取消按钮
})
```

### 隐藏 Modal

```
Modal.hide()
```

## 特性

* 每次仅显示一个 Modal。若重复调用，第二个自动覆盖第一个。
